COMPUTER LANGUAGES

Table to display the list of computer languages in columns:


SERIAL NO. LANGUAGE NAME CREATED BY YEAR OF STARTING PURPOSE
1 C Dennis M. Ritchie 1969 C is a powerful general-purpose programming language. It can be used to develop software like operating systems, databases, compilers, and so on.
2 C++ Bjarne Stroustrup 1985 C++ is a powerful general-purpose programming language. It can be used to develop operating systems, browsers, games, and so on. C++ supports different ways of programming like procedural, object-oriented, functional, and so on.
3 JAVA James Gosling 1991 Java is a powerful general-purpose programming language. It is used to develop desktop and mobile applications, big data processing, embedded systems, and so on.Java is one of the most popular programming languages.
4 PYTHON Guido van Rossum 1991 Python is a general-purpose language, meaning it can be used to create a variety of different programs and isn't specialized for any specific problems.
5 HTML Tim Berners-Lee 1993 HTML is the code that is used to structure a web page and its content. Contents could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables
